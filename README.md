# Happy-New-Year-2024

This project features an interactive fireworks animation that generates colorful bursts when clicked. 
The animation is implemented using a combination of particles and fireworks objects, creating an aesthetically pleasing display.

## Code Structure

- **index.html**: The HTML file containing the canvas element.
- **script.js**: The JavaScript file with the animation logic.
- **style.css**: The CSS file for basic styling.
